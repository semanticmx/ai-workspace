# TODOs Directory

Store and manage AI-generated TODO lists and task tracking.

## Structure

- **active/** - Current tasks you're working on
- **completed/** - Archive of completed tasks (for reference and review)
- **templates/** - Reusable TODO templates for common workflows

## File Naming Convention

Suggested format: `YYYY-MM-DD-description.md`

Examples:
- `2024-01-17-daily-tasks.md`
- `2024-01-17-project-alpha.md`
- `weekly-review-template.md` (in templates/)

## Example TODO Format

```markdown
# TODO: Project Name
Date: 2024-01-17
Status: In Progress

## High Priority
- [ ] Task 1
- [ ] Task 2

## Medium Priority
- [ ] Task 3
- [ ] Task 4

## Low Priority
- [ ] Task 5

## Notes
- Important context or decisions
```

## Tips
- Review active TODOs daily
- Move completed TODOs to completed/ folder monthly
- Use checkboxes for easy tracking
- Add context and notes for complex tasks
